# 01-local-ssh-control


## Controls

The tank itself is controlled with the following buttons.

> **NOTE** You must have an active SSH session **with keyboard focus** to the pi running the `local_tank.py` for the controls to work.

|    Action    | Button |
|:------------:|:------:|
|    Forward   |    W   |
|    Reverse   |    S   |
|  Rotate Left |    A   |
| Rotate Right |    D   |
| Beep         |    B   |
